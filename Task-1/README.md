## Description
This project is an interactive online quiz website where users can answer multiple-choice questions and get their score at the end. The application runs completely in the browser and does not require any backend.

## Tech Stack
- HTML  
- CSS  
- JavaScript  

## Features
- Start quiz screen  
- Multiple-choice questions  
- Automatic score calculation  
- Final result display  
- Restart quiz option  

## How to Run
1. Download or clone this repository
2. Open the `index.html` file in any modern web browser
3. Start the quiz
